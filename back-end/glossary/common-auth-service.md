# common-auth-service 용어

| 용어 | 설명 |
|------|------|
| ApplicationToken | 서비스 간 인증을 위한 애플리케이션 레벨 API 시크릿 키 |
| DeliveredCustomer | Delivered 플랫폼 고객 ID (customerId, userName) |
| GlobalShipAdmin | GlobalShip 플랫폼 관리자 (adminId, userName, email) |
| GlobalShipCustomer | GlobalShip 플랫폼 고객 (customerId, userName, email) |
| ProductProxyUser | ProductProxy 서비스 인증 사용자 (SELLER/ADMIN 타입) |
| TokenUser | Token 엔티티를 User 인터페이스로 래핑한 인증 객체 |
