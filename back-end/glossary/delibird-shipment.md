# delibird-shipment 용어

| 용어 | 설명 |
|------|------|
| B2bRequestShipment | B2B 라벨 요청 (배송사, 주문번호, 중량, 주소, 라벨 URL) |
| CarrierPattern | 송장번호-배송사 매칭 패턴 규칙 (접두사, 접미사, 길이) |
| CarrierRegex | 송장번호 배송사 감지용 정규식 규칙 |
| Custom | 통관 신고 정보 (내용물 타입, 포장 타입, 배송 타입, 수출자) |
| CustomItem | 통관 신고 개별 항목 (수량, 이름, HS 코드, 원산지, 가격) |
| CustomsExporter | 수출자 상세 (통관번호, 사업자번호, 회사명) |
| ExportDeclaration | 수출 통관 신고 (사업자 등록, 수출자 이름/주소) |
| Integration | 외부 시스템 연동 설정 (API 토큰) |
| Option | 배송 옵션 (반송 여부, 인코텀즈, 라벨 일자, 메모) |
| Package | 배송 물리적 패키지 (치수, 중량) |
| Reference | 주문/배송 참조 정보 (배치 ID, 주문 ID, 주문번호, 주문일) |
| RequestShipment | 배송 라벨 요청 (배송사, 서비스, 상태, 송장번호, 주소) |
| ShipHeroRequest | ShipHero 연동 요청 (주문, 배송사, 주소, 라벨) |
| ShipmentErrorLog | 배송 처리 실패 에러 로그 |
| ShipmentTransaction | 배송 관련 금융 거래 (송장번호, 금액, 배송사) |
